## 使用OpenCV进行人脸识别

1. 删除model文件夹下的h5文件;
2. 先运行faceTrain.py文件训练生成网络;
3. 再运行faceRecognition.py进行人脸识别.

## 运行环境:
1. python 3.6.5
2. TensorFlow 1.5.0
3. Keras 2.2.4
4. opencv-python 4.1.0.25
5. pip3 install numpy scipy matplotlib scikit-learn
